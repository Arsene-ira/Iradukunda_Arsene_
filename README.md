# OOP Programming Exercises
**Student ID:** 27206

## Description
This repository contains 10 Object-Oriented Programming exercises demonstrating:
- Inheritance (10-level class hierarchy)
- Encapsulation (private fields, public getters/setters)
- Custom exception handling
- Input validation
- Real-world system implementations

## Projects Included
1. **Hospital Management System** - Patient records and billing
2. **School Management System** - Student records and grading
3. **Employee Payroll System** - Salary calculations
4. **Hotel Reservation System** - Booking management
5. **Vehicle Rental System** - Rental invoicing
6. **Banking System** - Account and loan management
7. **Real Estate Management** - Property transactions
8. **Library Management System** - Book borrowing and fines
9. **Airline Ticketing System** - Flight bookings
10. **Online Shopping System** - E-commerce orders

## How to Run
```bash
# Compile any system
javac QuestionX_SystemName/SystemName.java

# Run
java QuestionX_SystemName.SystemName
```

## Technologies
- Java
- Scanner for input
- Custom exception handling
- LocalDateTime for timestamps
```
